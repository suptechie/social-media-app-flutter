# social_media_app

A social media application developed using Flutter and GetX state management.

## Features

- Login
- Registration
- Get Profile Details
- Update Profile Details
- Upload Profile Picture
- Delete Profile
- Add Post
- Get Posts
- Like/Unlike Post
- Update Post
- Delete Post
- Add Comment
- Get Comments
- Like/Unlike Comment
- Delete Comment
